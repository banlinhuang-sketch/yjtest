# 浏览器回归清单

## 启动准备

1. 执行 `npm.cmd run dev:all`
2. 打开 [http://127.0.0.1:4177](http://127.0.0.1:4177)
3. 确认本地 API 运行在 `http://127.0.0.1:8787`

## 核心链路

### P0 登录

- 正确账号密码登录成功
- 错误密码登录失败并提示错误
- 刷新页面后仍能恢复登录态

### P1 工作台

- 首屏用例列表成功加载
- 搜索、范围筛选、优先级筛选生效
- 生成草案后新用例插入列表顶部
- 生成后自动打开右侧编辑区
- 克隆用例后生成新的草稿条目

### P2 用例详情

- 从工作台或审核中心进入详情页
- 修改基础字段、结构化步骤并保存成功
- 必填项缺失时阻止保存
- 有未保存修改时返回会弹确认

### P3 审核中心

- `待审核 / 已处理` 切换正常
- 关键词、范围、优先级、提交人、时间筛选生效
- 退回草稿必须填写审核意见
- 审核通过后条目从待审核流转到已处理

### P4 导出中心

- 多条件筛选生效
- 创建导出任务后按钮进入 loading
- 导出成功后可下载文件
- 导出失败时展示明确错误提示

### P5 知识基线

- 进入页面后能拉到知识源列表
- 点击同步全部资源会触发刷新
- 无数据时展示空状态

### P6 状态模板

- 能切换并预览以下状态：
  - 登录失效
  - 无数据
  - 筛选无结果
  - 无权限
  - 系统异常
  - 网络异常

## 自动化辅助

- 执行 `npm.cmd run smoke:api`
- 确认 smoke 覆盖：
  - 登录
  - `auth/me`
  - 列表拉取
  - 草案生成
  - 保存
  - 审核
  - 导出
  - 知识源读取

## 通过标准

- `npm.cmd run lint` 通过
- `npm.cmd run build` 通过
- 浏览器主链路无阻塞项
- smoke API 脚本通过
